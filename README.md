# UUID 生成器

> UUID v4 批量生成器，支持无连字符格式，一键复制，零依赖纯前端

## 功能概览

- **UUID v4 随机生成，标准 36 字符格式 + 32 字符无连字符格式**
- **批量生成：1-100 条批量生成，一次复制全部**
- **生成数量统计 + 去重保证（同一批次内不重复）**
- **额外功能：随机字符串生成（自定义长度和字符集）**
- **一键复制全部结果，支持单独点击复制单条**

## 操作说明

1. 打开 index.html（浏览器直接运行，无需服务器）
2. 按界面引导输入/操作
3. 结果实时预览，支持一键复制导出

## 技术栈

- **HTML5**：语义化结构
- **CSS3**：白色极简主题，Flex/Grid 布局
- **Vanilla JavaScript**：纯原生代码，零依赖
- **ES6+**：sync/await、箭头函数、模板字符串
- **localStorage**：偏好设置持久化
